Big Data Management and Processing: Processing Data Stream for Streaming and Static Data Visualisation

In Year 3, I had simulated real-time streaming of climate and hotspot data using real-life fire and weather data. This was done by implementing multiple Apache Kafka producers to simulate the real-time streaming of data which is then processed by Apache Spark Streaming client and then inserted into MongoDB. Full specifications of the project can be found here (All rights belong to Monash University). I then performed two types of data visualisations, streaming and static data visualisations.
